# LuminAI - TCC

Repositório central do Trabalho de Conclusão de Curso. 

Este repositório contém a documentação (proposta, artigos, monografia) e o código-fonte do **LuminAI**, uma aplicação web focada em leitura ativa. O sistema implementa uma arquitetura de RAG (Retrieval-Augmented Generation) Multimodal, utilizando Visão Computacional e modelos de linguagem para processar simultaneamente textos e recortes espaciais (gráficos e diagramas) em documentos PDF.

## Estrutura do Repositório
- `/docs`: Documentação, rascunhos e arquivo final da monografia.
- `/src`: Código-fonte da aplicação web e da API de inteligência artificial.
